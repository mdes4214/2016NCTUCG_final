Control:
w, s: rotation_z
a, d: rotation_y
t, g: height
f, h: width
j, k: light intensity
u, i: roughness
z: two side
q: spin toggle
o, p: spin speed
[: toggle of changing mode while spinning
]: toggle of scaling while spinning
e: reset
m: change mode (4-edge, 8-edge, 12-edge polygon)
esc: exit

Color Control:
1: reset diffuse color
2: set diffuse color R=0 
3: set diffuse color G=0 
4: set diffuse color B=0 
5: reset specular color
6: set specular color R=0 
7: set specular color G=0 
8: set specular color B=0 
0: reset all color
